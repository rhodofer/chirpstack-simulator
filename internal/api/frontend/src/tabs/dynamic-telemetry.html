Dinamik Telemetri

Cihazların statik veri göndermesi yerine zaman serisi halinde değişen fiziksel sensör verileri üretin.

💡 Betik Çalıştırma Kılavuzu

Betik, sunucu tarafında her veri gönderme adımında yürütülür ve return [byte1, byte2, ...] şeklinde bir **byte dizisi (array)** veya **hexadecimal dizgi (string)** (örn: return "010203") döndürmelidir.
  • Kullanılabilir Değişkenler: fPort (port), fCnt (uplink sayacı), devEUI (cihaz EUI), deviceName (isim).
  • Matematiksel Fonksiyonlar: Standart JS metotları kullanılabilir: Math.sin(), Math.random(), Math.round(), vb.
