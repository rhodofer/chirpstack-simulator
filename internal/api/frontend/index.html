Simülasyon Çalıştırma Kılavuzu

  • Ağ Yapılandırması: Sağ çekmeceyi açarak simülasyon ayarlarını düzenleyin ve "Ayarları Kaydet" ile kaydedin. Yeni bir ağ kurmak için üst barda bulunan "Yeni Oluştur" sihirbazını kullanabilirsiniz.
  • Simülasyonu Başlatma: Üst menüdeki yeşil "▶ Başlat" butonuna tıklayın. Veritabanındaki tüm organizasyonlar aynı anda paralel çalışmaya başlayacaktır.
  • Ağ Katılımı (OTAA): Cihazlar otomatik olarak ChirpStack üzerinde OTAA katılım isteği gönderir ve onaylandığında veri göndermeye başlar.
  • Canlı İzleme ve Filtreleme: "Konsol" sekmesinden veri akışını izleyin. Yoğun log akışında arama kutusunu kullanarak spesifik bir cihazı (örn: sim-dev-1) filtreleyin.
  • Süre Güncelleme: Çalışma sırasında "Cihazlar" listesinden veri sıklığı ayarını (örn: 4 dk) değiştirin. Sistem simülatörü otomatik olarak durdurup yeni ayarla kesintisiz yeniden başlatacaktır.
  • Simülasyonu Durdurma: İstediğiniz zaman üst menüdeki kırmızı "🛑 Durdur" butonuna basarak tüm cihazları pasif konuma çekebilirsiniz.

Doğrudan Kullanılabilir Test Betikleri

ChirpStack simülatörü dinamik JS motoru ile uyumlu, veri dönüşümlerini açıklayan pratik örnekler.

1. Sıcaklık Dalgalanması (Byte Array)

Cihazın ısınmasını ve soğumasını sinüs dalgası ve gürültü ekleyerek gerçekçi simüle eder. Sıcaklık 10 ile çarpılarak LSB/MSB formatında 2 byte olarak döndürülür.

// Global degiskenler: fPort, fCnt, devEUI, deviceName
// 20.0 + sin dalgasi ile sicaklik dalgalanmasi (-5 to +5) ve gurultu
var baseTemp = 20.0;
var wave = Math.sin(fCnt * 0.05) * 5.0;
var noise = (Math.random() - 0.5) * 0.5;
var currentTemp = baseTemp + wave + noise;

// LSB/MSB formatinda 16-bit isaretli tamsayi (int16 * 10)
var scaledTemp = Math.round(currentTemp * 10);

// Negatif degerleri guvenli bir sekilde donusturmek icin bit maskeleme
var tempByte1 = scaledTemp & 0xFF;        // LSB
var tempByte2 = (scaledTemp >> 8) & 0xFF; // MSB

// Goja motoru icin byte dizisi dondurur
return [tempByte1, tempByte2];

2. GPS Rota Simülasyonu (Hex String)

Başlangıç noktasından (örn. İstanbul) itibaren deterministik olarak kuzeydoğu yönünde ilerleyen bir rotayı simüle eder. Koordinatları 100.000 ile çarparak 4'er byte işaretli tamsayıya çevirir ve Hex dizesi olarak döndürür.

// Istanbul koordinatlari merkezli baslangic
var startLat = 41.0082;
var startLng = 28.9784;

// Her adimda (fCnt) hafif kuzeydoguya hareket simülasyonu
var currentLat = startLat + (fCnt * 0.0001);
var currentLng = startLng + (fCnt * 0.00015);

// 100,000 ile carpilarak 32-bit integer donusumu
var latVal = Math.round(currentLat * 100000);
var lngVal = Math.round(currentLng * 100000);

// 32-bit sayiyi hex formatina donusturen yardimci fonksiyon (big endian)
function toHexInt32(val) {
    if (val < 0) {
        val = 0xFFFFFFFF + val + 1; // 2's complement
    }
    var hex = val.toString(16);
    while (hex.length < 8) {
        hex = "0" + hex;
    }
    return hex;
}

// 8-byte hex string (4 byte Lat + 4 byte Lng)
var payloadHex = toHexInt32(latVal) + toHexInt32(lngVal);

// Goja motoru icin hex string dondurur
return payloadHex;

3. Akıllı Sayaç & Pil Seviyesi (Byte Array)

Zamanla artan enerji tüketim endeksini (32-bit) ve her adımda yavaşça azalan pil seviyesini (%100'den geriye) byte dizisi olarak döndürür.

// Baslangic endeksi (10,000 Wh) + adim basina ~5 Wh artis
var baseEnergy = 10000;
var currentEnergy = baseEnergy + (fCnt * 5);

// Pil seviyesi: baslangicta %100, her 100 adimda %1 azalma (en az %1)
var batteryLevel = 100 - Math.floor(fCnt / 100);
if (batteryLevel < 1) {
    batteryLevel = 1;
}

// 32-bit unsigned integer donusumu (4 byte - Big Endian)
var energyB3 = (currentEnergy >> 24) & 0xFF;
var energyB2 = (currentEnergy >> 16) & 0xFF;
var energyB1 = (currentEnergy >> 8) & 0xFF;
var energyB0 = currentEnergy & 0xFF;

// Goja motoru icin 5 byte'lik dizi dondurur
return [energyB3, energyB2, energyB1, energyB0, batteryLevel];

4. Çoklu Sensör Telemetrisi (Hex String)

Sıcaklık (1 byte), bağıl nem (1 byte) ve fan çalışma durumunu (1 byte, boolean) birleştirip 3 byte Hex formatında döndürür.

// Sicaklik simülasyonu (0 - 50 araligi)
var temp = 22 + Math.floor(Math.sin(fCnt * 0.1) * 8);

// Nem simülasyonu (40% - 80% araligi)
var humidity = 60 + Math.floor(Math.cos(fCnt * 0.08) * 15);

// Sicaklik 28 derecenin uzerindeyse fan aktif (true/1) aksi halde pasif (false/0)
var fanStatus = (temp > 28) ? 1 : 0;

// Byte degerlerini tek basamakli olmamasi icin hex string formatina donusturen yardimci fonksiyon
function toHexByte(val) {
    var hex = (val & 0xFF).toString(16);
    return hex.length === 1 ? "0" + hex : hex;
}

// 3 byte hex string
var payloadHex = toHexByte(temp) + toHexByte(humidity) + toHexByte(fanStatus);

// Goja motoru icin hex dondurur
return payloadHex;
